# Weather AI Assistant

## Opis projektu

Weather AI Assistant to chatbot webowy pomagający użytkownikowi dobrać odpowiedni ubiór do aktualnej pogody.

Użytkownik wpisuje informacje o pogodzie, a system analizuje dane i rekomenduje odpowiedni strój, dodatki oraz ochronę przed deszczem lub zimnem.

## Technologie

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Responsive Web Design

## Funkcjonalności

* dynamiczny chatbot
* analiza danych użytkownika
* rekomendacja ubioru
* sugestie dodatków
* responsywny interfejs
* animacje wiadomości
* automatyczny scroll
* wysyłanie wiadomości klawiszem ENTER

## Uruchomienie projektu

1. Pobierz pliki projektu
2. Otwórz plik `index.html`
3. Korzystaj z chatbota

## Przykładowe rozmowy

**Użytkownik:**
„Jest 7 stopni i pada deszcz”

**Bot:**
„Załóż kurtkę przejściową lub bluzę oraz długie spodnie. Nie zapomnij o parasolu i wodoodpornych butach.”
